This project gives demo of how to canvas element in QML to generate graph lines 
This gives demo of how to take mouse inputs from user and do some processing and show selected data. 

This project draws graph lines using random points. 
User can select portion of the graph to see the content in zoomed fashion


Project is split in multiple components such as 
1. wavehighlighter : used to handle user inputs to show certain area of the graph in highlighted manner
2. waverectangle : main component to draw the graph, it accepts points as input and works on it
3. inspectionpopup : this will be parent for showing zoomed in area of the graph
